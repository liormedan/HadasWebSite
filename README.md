# Hadas Dror Home Site

אתר בית אישי להדס דרור, בנוי עם Next.js ומוכן לפריסה ב־Vercel.

## הרצה מקומית

```bash
npm install
npm run dev
```

האתר יעלה בכתובת:

`http://localhost:3000`

## פריסה ל־Vercel

1. יוצרים ריפו ב־GitHub ומעלים את הקבצים.
2. מתחברים ל־[Vercel](https://vercel.com/) עם אותו משתמש.
3. Import Project ובוחרים את הריפו.
4. מאשרים Deploy.

## התאמות מהירות שכדאי לעשות

- החלפת אימייל ב־`app/page.js`.
- החלפת מספר וואטסאפ ב־`app/page.js` (`https://wa.me/...`).
- עדכון טקסטים, ביוגרפיה וכותרות לפי הצורך.
